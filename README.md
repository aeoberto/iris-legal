# IRIS

IRIS (Interactive Rendering Interface System) is a smart LED display platform designed for real-time status visualization and device interaction.

The system uses an ESP32-based LED matrix controlled via an iOS application, allowing users to display availability, status, and system information in real time.

## Features
- Real-time status display
- MQTT-based communication
- iOS app control
- LED matrix rendering
- Designed for low-latency operation

## Hardware
- ESP32 (Matrix Portal S3)
- HUB75 LED Matrix
- Optional IMU (ICM-20948)

## Status
This project is currently under active development.
